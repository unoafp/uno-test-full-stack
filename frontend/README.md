# Frontend - Memory Game UI

## Description

Frontend application for the Memory Game built with [Next.js](https://nextjs.org/).

## Prerequisites

- Node.js
- Backend API running (see root or backend README)

## Configuration

1. If environment variables are needed, copy the example file (currently none required but good practice):
   ```bash
   cp .env.example .env.local
   ```

## Installation

```bash
npm install
# or
yarn install
```

## Running the app

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
